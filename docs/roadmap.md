# Roadmap

## Phase 1

Runnable local MVP with mock-first AI companion chat, Elden Ring process status, local knowledge search, and JSONL memory.

## Phase 2

- Push-to-talk recording with real STT.
- TTS provider integrations.
- Live2D avatar layer.
- RAG-backed Elden Ring knowledge base.
- OCR or vision-assisted game state context.
- Multi-game and multi-character registries.
- Long-term memory and player preference modeling.
